HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The id attribute is a way to identify an element and select it
  • The class id works similar to the id attribute
  • Unordered bulleted list or ordered lists can be created with the ul or ol element

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • the period precending the "card" designated this a class selector
  • Can assign colors as semantic reference or by hexadecimal code
  • Must link the CSS file to the HTML through the HTML link element

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main will check to make sure that your branch matches the main branch
  • git push origin followed a branch name will push the local branch to the the remote repository origin
  • git status will show the state of the working directory and the staging area

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Code is executed top to bottom
  • A function must be called in order for it to be executed
  • Parameters can be passed to the function inside of the parathesis in the function declaration
